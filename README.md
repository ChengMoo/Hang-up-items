支持：开启了Hyper-V的电脑，手机

网心云共享闲置，可以自定义设置共享的磁盘大小和核数和内存，也可以挂载一下阿里云的云盘，不用自己的固态或机械盘

（国内的平台，直接提现到支付宝或银行卡）

（这个不能挂云服务器，云服务器无法开启Hyper-V虚拟化）

如果挂载更大的磁盘或上行更高，收入一天估计几块钱吧，当然你地区是它需要的地区收入更高，我的不是急需地区。
